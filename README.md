# Port Scanner (Python)

This project scans open ports on a target IP address using multithreading for faster performance.

# Python Port Scanner

Python ile yazılmış basit bir port tarayıcı.  
Belirlediğiniz IP veya domain üzerindeki açık portları listeler.

## Kullanım

1. Terminal ya da komut satırında python port_scanner.py çalıştır.
2. Hedef IP veya domain girin.
3. Açık portlar ekranda listelenecek.
